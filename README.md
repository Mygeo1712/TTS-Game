
# TTS MASTER

Aplikasi Teka Teki Silang Fullstack.

## Instalasi
1. Buat database `tts_db` di PostgreSQL.
2. Jalankan query di `db.sql`.
3. Sesuaikan `db.js` (password).
4. `npm install express pg`
5. `node server.js`
6. Akses `http://localhost:3000`.

## Fitur
- **Admin**: Input 5-15 kata, auto generate grid, preview, publish.
- **Player**: Grid interaktif, keyboard support, timer, deteksi menang.
- **Teknologi**: Node.js, PostgreSQL, React (ESM).
